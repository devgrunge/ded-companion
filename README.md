# ded-companion

## Introduction

This is a board game assistant, Node.Js with Fastify

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js
- TypeScript

### Installing

1. Clone the repository:

 ```bash
   git clone https://github.com/devgrunge/ded-companion.git
   ```
 ### Running
 1. Run this script on terminal: 

   ```bash
     npm run dev
   ```
